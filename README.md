# 三国志文字版 — 战局实况

## 项目简介
三国志文字版的实时战局展示网站。GM（主持人）将 AI 生成的回合内容粘贴后一键发布，系统自动提取关键结构化数据，AI 原文整体展示在战局动态，所有玩家页面自动同步更新。

当前版本：**v2.7.9**（2026-05-08）

---

## 已完成功能

### 战局总览（玩家端）
- **回合标题条**：显示当前回合编号
- **📜 战局动态**：直接展示 AI 原文全文，智能排版 + 关键词高亮
- **🗺️ 势力地图 v12（v2.5 最终版）**：
  - 60 座城池（幽/冀/并/青/司隶/雍凉/兖豫/徐州/荆襄/扬州/益州/南中）
  - 等大 flat-top 六边形固定布局，7 种地形底色
  - 城池弹窗（v2.5）：驻将 + 分兵种兵力展示
- **⚔️ 战斗结算**：三栏卡片（攻方 | 结果徽章 | 守方），含伤亡数值
- **👥 三方势力卡**：玩家名、主城、金/粮/兵/士气、武将列表、城池数量
- **📊 本回合收支详情（v2.7.9）**：
  - pcard 战旗风格三栏卡片（深色 + 左色条）
  - 资源总变化 pills（固定顺序：金→粮→兵→民心）
  - 收支明细表（分项顺序固定：产出→维护→季度→明账→府库→贸易→事件）
  - 季度结算芯片
  - **府库调度**（原"暗账"，右对齐数字 + desc 截断 ≤8 字）
  - **兵种变动**（每城独立 section，标题格式「兵种△ 城名」）
  - **情报动向**（两栏分类：⚔️ 战报回响 / 🏅 麾下动态）
  - **数据校验警告**：收支合计与总变化不符时高亮提示
  - NPC 天下动态横幅（全宽，城市状态 + 野外动向）
- **📜 历史回合**：可折叠的历史回合列表

### GM 端
- 粘贴 AI 原文 → 预览 → 一键发布
- 支持双段格式：剧情区 + `════ DATA ════` 数据区
- 兼容旧版格式，撤销/清空功能，30 秒自动云同步轮询

## 数据格式规范（v2.7.9）

### [变动] 块支持的锚点
| 锚点 | 说明 |
|------|------|
| `收支△` | 资源收支明细（格式 A/B 均支持） |
| `府库△` | 府库调度条目（原`暗账△`，新旧均识别） |
| `季度△` | 季度结算，同行内容：`季度△金-280,粮-420` |
| `情报△` | 情报列表，同行内容：`情报△陈留粮市继续运转` |
| `兵种△城名:规格` | 兵种变动，示例：`兵种△长安:骑+500,步-200` |
| `NPC 城名状态△内容` | NPC 城市状态事件 |
| `野外△内容` | 野外动向事件 |

### 城池格式（v2.5）
```
城名(守将/守将2|兵种:数量,兵种:数量)   ← 标准格式（英文括号/斜杠/逗号）
城名(守将)                              ← 旧格式兼容
```
> 非标准符号（中文括号/全角逗号）将触发 console.warn

### 武将状态白名单
`健康` / `疲劳` / `受伤` / `患病` / `阵亡`（其他值视为健康并 warn）

| 路径 | 说明 |
|------|------|
| `/` (index.html) | 主战局页面（玩家端 + GM 端同页） |

---

## 数据模型

### 表：`sanguo_rounds` v2
| 字段 | 类型 | 说明 |
|------|------|------|
| id | text | UUID，系统自动生成 |
| round | number | 回合编号 |
| round_title | text | 回合标题（可选） |
| raw_content | text | GM 原始粘贴文本（用于实时重解析） |
| players_json | text | 玩家数组 JSON |
| battles_json | text | 战斗结算数组 JSON |
| city_ownership_json | text | 城池归属 Map JSON（含 holder/troops） |

### change 对象结构（parser.js 输出）
```js
{
  slot: '甲',           // 玩家槽
  resources: {},        // 总资源变化 {金,粮,兵,民心}
  breakdown: {},        // 收支明细 {金:{items:[{label,val}],total}}
  darkItems: [],        // 府库调度 [{desc,entries:[{res,val}]}]
  seasonal: [],         // 季度结算 [{res,val}]
  intel: [],            // 情报列表 [string]
  troopChanges: [],     // 兵种变动 [{cityName,entries,isDelta}]
  warnings: [],         // 校验报警 [string]
}
// result.__npc = [{type:'npc'|'wild', city, desc}]
```

## 技术栈
- **前端**：HTML5 / CSS3 / Vanilla JS（无框架）
- **字体**：Google Fonts（Noto Serif SC、Noto Sans SC）
- **地图**：纯 SVG，flat-top 六边形坐标系手动布局
- **数据存储**：RESTful Table API（云端 30 秒轮询同步）
